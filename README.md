pydocx_ja_font
====

python用のwordを扱うためのライブラリのpython-docxを利用するときに、日本語用フォントを使いたいときに用いるクラスです。


## Description 説明
日本語フォントを選択して、文章の入力、表の作成を行うことができます。

## How to 使い方
1. プログラムと同じディレクトリにtmp.docxという空のwordを作成します。
1. 表を使いたいときは、そのwordファイルを開いて表を挿入します。表のスタイルは「表(格子)」を選択します。そして、挿入した表を削除して上書き保存します。この手順は作製したwordのファイルの中に「Table Grid」という表のスタイルの情報を入れ込むためです。一度このスタイルを使えば、python側から自由に使用することができるようになります。
1. 後はサンプルに従って、自由に書いてください。
